# Network-Graphs-of-Game-of-Thrones-The-TV-Series-&-Books
Network Graph Visualizations of the Game of Thrones TV series & Book collection

**Contents**
- About
- Datasets
- Classes
- Executables
- Outputs

**About**</br>
I have constructed network graphs of the interactions between characters on the game of thrones TV series as well as the book collection on which the TV series is based upon. I have used Gephi to create the graph visualizations.

*To execute the program run the main.py script* <br/> 
*To create the hypergraphs run the hypergraph.py script*

**Datasets**</br>
The primary data and the constructed data are stored in the data directory. The primary dataset was taken from "Network of Thrones" website.

**Classes**
- **join_edges**: Contains Class JoinEdges to create the edges csv dataset for the network graph.   

**Executables**
- **main**: Initializaes the the global variables and data paths, Creates the Nodes and Edges csv files  for incremental cumulative datasets to be used as input for Gephi to create dataset Visualizations.
- **hypergraph**: Creates the nodes and edges csv files for the hypergraph networks.
